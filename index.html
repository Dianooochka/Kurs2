• git status-показує поточні зміни в коді
  • git add.-додає всі файли до майбутнього коміту
  • git status-можна подивитись, які файли були змінені
  • git commit -m "add index file"-зберегти що саме десь змінилось
  • git log-історія коммітів
  • git push origin main-відправка закоммічених файлів до репозиторію
  • git remote-які репозиторії привʼязані до нашої папки
  • git checkout -b name-дозволяє перейти на іншу гілку
